Installation
============

You can install the package as follows:

.. tab-set::

    .. tab-item:: latest version

        Using pip

        .. code-block:: console

            (.venv) $ pip install git+https://github.com/mipt-srf/grain-processor

        or uv

        .. code-block:: console

            $ uv add git+https://github.com/mipt-srf/grain-processor

    .. tab-item:: latest PyPI release

        Using pip

        .. code-block:: console

            (.venv) $ pip install grain-processor

        or uv

        .. code-block:: console

            $ uv add grain-processor
